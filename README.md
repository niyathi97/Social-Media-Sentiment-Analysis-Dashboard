# 📊 Social Media Sentiment Analysis Dashboard

## 🚀 Project Overview

This project is a Machine Learning-based dashboard that analyzes social media text and classifies it into Positive or Negative sentiment using Natural Language Processing (NLP).

## 🎯 Objective

To help businesses understand customer opinions, feedback, and brand perception by analyzing large volumes of social media data automatically.

## 🧠 Features

* Text preprocessing using NLTK (tokenization, stopword removal, cleaning)
* TF-IDF feature extraction for text representation
* Logistic Regression model for sentiment classification
* Real-time sentiment prediction
* Interactive dashboard built using Streamlit

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* NLTK
* Scikit-learn
* Streamlit

├── README.md
├── requirements.txt

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   streamlit run app/app.py

3. Open in browser:
   http://localhost:8501
